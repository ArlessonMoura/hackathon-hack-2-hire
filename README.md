# IDP Enterprise Landing Page

## 🚀 Visão Geral

Está é uma landing page de apresentação da Plataforma Leitur.AI, uma solução de Intelligent Document Processing (IDP) construída com Amazon Bedrock e arquitetura serverless AWS.
Nossa plataforma, desenvolvida durante o hackathon Hack2Hire - Escola da Nuvem/AWS, já se encontra pronta para produção. Com ela obtvemos o 3° lugar na competição e os melhores apontamentos dos jurados nos quesitos maturidade arquitetural, expertise em AWS e visão de negócios.

## 📁 Estrutura do código desta landing page

```
landing-page/
│
├── index.html              # Arquivo HTML principal
│
├── css/
│   ├── variables.css       # Propriedades customizadas CSS e design tokens
│   ├── reset.css           # Reset CSS moderno
│   ├── layout.css          # Estilos de layout e navegação
│   ├── components.css      # Estilos de componentes reutilizáveis
│   ├── sections.css        # Estilos específicos de seções
│   ├── animations.css      # Estilos de animação e transição
│   └── responsive.css      # Breakpoints responsivos
│
├── js/
│   ├── app.js              # Ponto de entrada principal da aplicação
│   ├── navigation.js       # Módulo de navegação com menu mobile
│   ├── counters.js         # Módulo de contadores animados
│   ├── charts.js           # Integração Chart.js
│   └── particles.js        # Módulo de animação de partículas
│
└── assets/
    ├── images/             # Assets de imagens
    ├── icons/              # Assets de ícones
    └── fonts/              # Arquivos de fontes
```
## 🛠️ Tecnologias

- **HTML5**: Marcação semântica com recursos de acessibilidade
- **CSS3**: CSS moderno com custom properties, flexbox e grid
- **JavaScript ES2023**: Arquitetura modular baseada em componentes
- **Bibliotecas**:
  - AOS (Animate On Scroll): Animações de scroll
  - Chart.js: Gráficos interativos para seções FinOps e Escalabilidade
  - Lucide: Biblioteca de ícones minimalista
